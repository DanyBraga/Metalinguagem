##  Descrição do Projeto

Este projeto consiste na criação de uma história em quadrinhos (HQ) com abordagem de **metalinguagem**, na qual o próprio processo de desenvolvimento do projeto é retratado dentro da narrativa.

A história acompanha três colegas de sala (as integrantes desse grupo) que se reúnem para desenvolver uma atividade acadêmica utilizando metodologias ágeis e ferramentas de **Inteligência Artificial Generativa**. Ao longo dos quadrinhos, as personagens discutem ideias, organizam tarefas e tomam decisões, refletindo o processo real do grupo.

Como elemento central de metalinguagem, o projeto utiliza IA para falar sobre o uso da própria IA em contextos criativos.
